Готовый файл:  build\libs\musicplayer-1.0.0.jar  → скопировать в папку mods.

───────────────────────────────────────────────────────────────
СПОСОБ 1 — Через Cursor / IntelliJ IDEA (БЕЗ GRADLE)
───────────────────────────────────────────────────────────────
Gradle ставить не нужно. IDE сама скачает всё и соберёт проект.

1. Откройте эту папку в Cursor или IntelliJ:
   File → Open → выберите папку "minecraft mod".

2. Дождитесь, пока IDE подгрузит проект (Gradle sync). Внизу будет
   прогресс "Loading Gradle project..." — подождите до конца.

3. Сборка:
   • Cursor / VS Code: Terminal → Run Task → "build" (или "gradle build"),
     или откройте панель Gradle и запустите build.
   • IntelliJ: меню Build → Build Project (Ctrl+F9).
     Или справа панель Gradle → musicplayer → Tasks → build → двойной клик.

4. Готовый JAR будет здесь:  build\libs\musicplayer-1.0.0.jar
   Перенесите его в папку mods вашей Minecraft (Fabric 1.21.1).

───────────────────────────────────────────────────────────────
СПОСОБ 2 — Через GitHub (БЕЗ GRADLE НА КОМПЬЮТЕРЕ)
───────────────────────────────────────────────────────────────
Сборка идёт на серверах GitHub, у вас Gradle не нужен.

1. Залейте этот проект на GitHub (создайте репозиторий и загрузите папку).

2. Откройте репозиторий на GitHub → вкладка Actions.
   Должен запуститься workflow "Build mod" (или запустите его вручную:
   Actions → Build mod → Run workflow).

3. После завершения: Actions → последний запуск → внизу блок
   "Artifacts" → скачайте musicplayer-1.0.0.jar.

4. Положите скачанный JAR в папку mods.

───────────────────────────────────────────────────────────────
СПОСОБ 3 — Через командную строку (если хотите использовать Gradle)
───────────────────────────────────────────────────────────────
• Установите Gradle: https://gradle.org/install/ или  winget install Gradle.Gradle
• В папке мода:  gradle build
• Или запустите  BUILD.bat
• JAR: build\libs\musicplayer-1.0.0.jar
